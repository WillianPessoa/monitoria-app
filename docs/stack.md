# Stack Tecnica - Monitoria App

Documento de referencia da stack definida para o projeto.

## Versoes definidas

- Python: 3.12
- Framework backend: Flask 3.0.3
- Banco de dados: MySQL 8.0
- Conector Python/MySQL: mysql-connector-python 9.0.0

## Justificativa

- Flask foi escolhido por simplicidade e curva de aprendizado adequada ao contexto da disciplina.
- MySQL atende ao modelo relacional do dominio e permite consultas SQL claras para auditoria.
- Queries SQL diretas foram escolhidas por transparência de comportamento e facilidade de revisao tecnica.
